# Review Translator 사용 가이드

> 지그재그 리뷰 요약 & 프랑스어 번역 테스트 도구

---

## 접속

**URL**: (배포 후 Railway URL)

---

## 화면 구성

| 영역 | 설명 |
|-----|------|
| **좌측 사이드바** | 프롬프트 버전 관리, 번역 히스토리 |
| **좌측 메인** | 데이터 입력, 프롬프트 편집, 번역 사전 |
| **우측 메인** | 결과 출력 (요약 → 번역 → 품질평가) |

---

## 사용 방법

### Step 1. 카테고리 선택

상단에서 **스킨케어** 또는 **메이크업** 선택
- 스킨케어: 피부 고민 요약 포함
- 메이크업: 만족도 + 리뷰 전반 요약만

---

### Step 2. CSV 파일 업로드

Redash에서 추출한 리뷰 CSV 파일을 드래그앤드롭

**필수 컬럼**:
- `content` 또는 `contents` 또는 `리뷰본문`: 리뷰 텍스트

**선택 컬럼**:
- `rating` 또는 `별점`: 평점 (만족도 계산용)
- `skin_concern` 또는 `피부고민`: 피부 고민 정보

> 파일 업로드 시 이전 데이터는 자동 초기화됩니다.

---

### Step 3. 데이터 확인

업로드 후 자동으로 표시되는 정보:
- 전체 리뷰 수
- 텍스트 리뷰 수
- 만족도 (별점 4~5점 비율)
- 피부 고민별 통계 (있는 경우)

**[리뷰 샘플 보기]** 클릭하면 실제 리뷰 내용 확인 가능

---

### Step 4. 요약 생성

**[1. 요약 생성]** 버튼 클릭

결과가 우측 **"1. 한국어 요약"** 영역에 표시됨

---

### Step 5. 번역

**[2. 번역]** 버튼 클릭

결과가 우측 **"2. 프랑스어 번역"** 영역에 표시됨

---

### Step 6. 품질 평가

**[3. 품질 평가]** 버튼 클릭

AI가 자동 평가 후 결과 표시:
- **PASS**: 품질 OK
- **REVIEW**: 검토 필요
- **FAIL**: 문제 발견

**최종 평가**에서 직접 Pass/Fail/Review 선택하여 저장

---

## 프롬프트 편집

좌측 하단 **프롬프트 편집** 영역에서 수정 가능

| 탭 | 용도 |
|----|------|
| 요약 | 리뷰 요약 생성 프롬프트 (카테고리별 다름) |
| 번역 | 한→프 번역 프롬프트 |
| 품질평가 | 번역 품질 체크 프롬프트 |

### 프롬프트 저장
1. 프롬프트 수정
2. 버전명 입력 (예: v1.1)
3. **[저장]** 클릭

### 저장된 버전 불러오기
1. 좌측 사이드바 → **프롬프트** 탭
2. 프롬프트 유형 선택
3. 원하는 버전 옆 **[적용]** 클릭

---

## 번역 사전

좌측 하단 **번역 사전** 영역

### 새 단어 추가
1. 한국어, 영어(선택), 프랑스어 입력
2. **[추가]** 클릭

### 등록된 단어 확인
**[등록된 단어 보기]** 클릭

### 사전 내보내기
**[📥 사전 CSV 다운로드]** 클릭

---

## 히스토리

좌측 사이드바 → **히스토리** 탭

- 이전 번역 결과 조회
- 필터: 전체 / Pass / Fail / Review / 미평가
- 각 항목 클릭하면 상세 내용 및 평가 변경 가능

---

## 초기화

| 버튼 | 동작 |
|-----|------|
| 상단 **[초기화]** | 현재 데이터만 초기화 |
| 하단 **[🔄 새 세션 시작]** | 모든 데이터 완전 초기화 |

---

## 주의사항

1. **CSV 컬럼명**: content, rating, skin_concern 권장
2. **새 파일 업로드 시**: 이전 요약/번역 결과 자동 삭제
3. **프롬프트 수정 후**: 반드시 버전 저장해두기
4. **품질 평가 후**: 최종 평가 선택해야 히스토리에 반영
